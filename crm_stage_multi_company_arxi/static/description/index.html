CRM Stage Multicompany Arxi

This module adds support for multi company on crm stage.


Demo video of RFID Endpoint Actions at Arxi YouTube Channel

  • This app allows you to define different States in the CRM module for multiple independent Companies.
  • When creating a State, the user can specify which Company it applies to.
  • By default, the selected company is the current one.
